# 📱 Telecommunications Plans | Comparative Analysis

## 📖 Project Overview

This project is a Power BI dashboard developed to analyze and compare telecommunications plans across major service providers in Malaysia.

The dashboard enables users to evaluate mobile plans based on pricing, value-for-money, bundled benefits, and overall plan features, helping consumers make more informed decisions when selecting a telecommunications package.

---

## 🎯 Objectives

- Compare mobile plans from different service providers.
- Identify the most budget-friendly plans.
- Analyze the average cost per GB of data.
- Highlight plans with the best bundled benefits.
- Provide a visual comparison of plan features and pricing.

## 🔍 Key Metrics

| Metric | Description |
|----------|-------------|
| Total Plans Available | Number of plans analyzed |
| Average Plan Price | Average subscription cost across plans |
| RM per GB | Cost efficiency of mobile data |
| Most Bundled Benefits | Provider offering the highest number of bundled features |
| Cheapest Plan | Lowest-priced plan available |
